SimplePHPCasClient 是让PHP程序访问CAS服务的一个SDK.

SimplePHPCasClient比[phpCAS](https://github.com/apereo/phpCAS)更好用一些

但是SimplePHPCasClient只实现了最基础的登录功能,后续会逐渐完善
